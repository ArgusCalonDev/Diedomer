# Diedomer
Repository ini untuk orang yang mau modifikasi game Diedomer buatan saya (open source!). Versi: **hfd-prototype-1.** 

**Cara modifikasi**: Klik tombol "fork" di bagian atas halaman repository. Lalu lanjutkan proses fork. Setelah berhasil, anda bisa **clone repo**, lalu **buka folder "diedomer" yang ada
di folder repository ke godot engine**. **PENTING SEKALI: godot anda sebaiknya memiliki versi 4.4 atau 4.4.1. Jika tidak maka proses akan berjalan tidak lancar**.

Sebelum itu, baca terlebih dahulu lisensi di file Lisensce yang ada di repository ini.
