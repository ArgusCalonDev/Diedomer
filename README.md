# Diedomer
**Diedomer** adalah game flappy-style yang memiliki rintangan rintangan seperti "lorong" di atas dan bawah, serta rintangan rintangan lainnya yang saya sebut sebagai 
"rintangan jumscare".

Repository ini untuk orang yang mau modifikasi game Diedomer buatan saya (open source!). Versi: **Alpha LE-1.** 

**Cara modifikasi**: Klik tombol "fork" di bagian atas halaman repository. Lalu lanjutkan proses fork. Setelah berhasil, anda bisa **clone repo**, lalu **buka folder "diedomer" yang ada
di folder repository ke godot engine**. **PENTING SEKALI: godot anda sebaiknya memiliki versi 4.4 atau 4.4.1. Jika tidak maka kemungkinan proses tidak berjalan lancar.**.

Sebelum itu, baca terlebih dahulu lisensi di file Lisensce yang ada di repository ini.
